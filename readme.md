## Getting Started

### Setup repo
```sh
git clone git@github.com:InfinityCodebase/edu-ai-hub.git
cd edu-ai-hub
```

### Setup env
```sh
chmod +x setup_env.sh
./setup_env.sh
```

### Install with docker

```sh
chmod +x install.sh
./install.sh
```
